# Artful - 신진 예술인을 위한 포트폴리오 & 팬 성장 플랫폼

> 아트풀(Artful)은 신진 예술인의 분산된 작품/이력을 하나의 아카이브형 포트폴리오로 통합하고,
> 팬 팔로우 기반의 지속 방문 구조와 B2B 캐스팅/협업 매칭까지 연계하는 예술인 전용 플랫폼입니다.

## 핵심 기능

| 기능 | 설명 |
|------|------|
| 듀얼 탭 | 피드(소비) + 포트폴리오(관리) 분리 정보구조 |
| 아카이브 포트폴리오 | 작품/이력/크레딧 통합 관리 |
| 제출용 링크 생성 | 원클릭 포트폴리오 공유 URL + QR 코드 |
| AI 프로필 분석 | 작품/이력/크레딧 구조화 데이터 변환 |
| 열람 분석 | 스크롤 뎁스 히트맵 + AI 코칭 |
| 팬 팔로우/응원 | 팔로우/응원/피드 리텐션 구조 |
| B2B 캐스팅 매칭 | 검증된 포트폴리오 기반 매칭 |

## 기술 스택

### 현재 (MVP)
- Vanilla JavaScript (Single Page Application)
- Tailwind CSS (CDN)
- LocalStorage 기반 데이터 관리
- QR Code Generator

### 프로덕션 (계획)
- **Frontend**: React (Next.js) + TypeScript + Tailwind CSS
- **Backend**: Node.js (Next.js API Routes)
- **Database**: PostgreSQL + Prisma ORM
- **Auth**: NextAuth.js (카카오/이메일)
- **Infra**: Vercel + Supabase
- **CI/CD**: GitHub Actions
